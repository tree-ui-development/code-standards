Editores de Código

Un gran editor de código puede hacer explotar la productividad de manera excepcional. Muchos desarrolladores prefieren editores de texto rudimentarios, mientras que otros prefieren poderosos entornos de desarrollo integrados (IDEs). Lo que verá a continuación es una lista general de algunas de las herramientas más conocidas. Sería imposible enumerarlas todas.

Aptana

Aptana Studio es un entorno de desarrollo de Ajax potente, gratis y de código abierto que corre independiente o dentro de Eclipse. Aptana Studio ofrece herramientas para Ajax incluyendo HTML, CSS, DOM y JavaScript, edición y depuración, además soporta PHP, Ruby on Rails, Adobe AIR y desarrollo para iPhone de Apple mediante plugins adicionales gratuitos. Incluye también integración completa con repositorios SVN para hacer committ, branching, tagging, merging y navegación de repositorios. [Linux, Mac, Windows]

Geany

Geany es un editor de texto que utiliza el kit de herramientas GTK2 con características básicas de un entorno de desarrollo integrado. Fue desarrollado para proporcionar un IDE pequeño y rápido, que tiene sólo unas pocas dependencias de otros paquetes. Soporta muchos tipos de archivo y tiene algunas características agradables. [Linux, Mac, Windows]

Notepad ++

Notepad ++ es un editor de código y reemplazo del Bloc de notas gratuito que soporta varios lenguajes de programación y que se ejecuta en el entorno de MS Windows. [Windows]

e TextEditor

E es un nuevo editor de texto para Windows, con potentes funciones de edición y bastantes habilidades únicas. Hace la manipulación de texto muy fácil y rápida y le permite enfocarse en su código mediante la automatización de todo el trabajo manual. Puede extenderse en cualquier lenguaje y dado que soporta los bundles de TextMate, le permite aprovechar una comunidad enorme y activa. e TextEditor. [Windows]

Edit Plus

EditPlus es un editor de texto, editor de HTML y editor para programadores hecho para Windows. Aunque puede servir como un buen reemplazo del Bloc de notas, también ofrece muchas características poderosas para programadores y creadores de páginas Web. [Windows]

Homesite

Homesite 5.5 proporciona un editor limpio, de sólo código para desarrollo web. Funciones avanzadas de código le permiten crear y modificar instantáneamente las etiquetas HTML, CFML, JSP y XHTML, mientras que las herramientas de productividad mejoradas le permiten validar, reutilizar, navegar y formatear su código más fácilmente. Configure Adobe (antes Macromedia) HomeSite para satisfacer sus necesidades mediante la extensión de su funcionalidad y personalización de la interfaz. [Windows]

TextMate

TextMate dice ser el "Editor que falta" para Mac OS X. Un editor de propósito general con una interfaz dispersa, su verdadero poder está en su extensibilidad. Incluye selección de columnas, macros grabables, snippets (fragmentos de código), auto cerrado de llaves y otros caracteres, historial del portapapeles, plegado de código, desencadenadores mediante tabulador, marcadores de posición de tabulaciones y escritura en espejo. Y eso sólo es para principiantes. Cualquier cosa que pueda realizarse mediante secuencias de comandos a través de la línea de comandos de Mac puede hacerse con comandos personalizados, permitiendo un muy alto grado de personalización y ampliación del conjunto de características. El formato de "bundles" de TextMate ha sido adaptado por muchos otros editores de código incluyendo el mencionado e-TextEditor. [Mac]

Espresso

Espresso fue creado por el mismo compañero que creó el innovador editor de CSS CSSEdit. Espresso incluye resaltado de sintaxis, plegado de código, completado de código, generador de esquema/navegador de documentos, proyectos, potentes características de búsqueda y capacidades de transferencia de archivos incluidas. Por último, tiene un poderoso conjunto de características "Sugar" que permite la creación de comandos personalizados y plugins. [Mac]

BBEdit

BBEdit es el abuelito de los editores de código de Mac para el desarrollo web. Incluye resaltado de sintaxis, herramientas de manipulación de texto excepcionalmente potentes, búsquedas en múltiples archivos, una API de secuencias de comandos, recortes de texto y amplias funciones de Automator de Mac. [Mac]

TextWrangler

El "hermano pequeño" gratuito de BBEdit, es un potente editor de texto sin formato con un conjunto masivo de funciones de manipulación de texto. Búsquedas, expresiones regulares, transformaciones de texto, resaltado de sintaxis y herramientas de navegación de código para una variedad de entornos de lenguaje. TextWrangler. [Mac]

Coda

Coda de Panic Software es un poderoso IDE con edición de código, terminal, gestión de archivos remotos y documentación de ayuda, todos integrados en una interfaz de usuario. Con el objetivo de ser un centro único para su flujo de trabajo de desarrollo web, también incluye integración de SVN y un nuevo generador de plug-ins con poderoso soporte de secuencias de comandos e importado de bundles de TextMate. Finalmente, también soporta recortes de código y edición multiusuario en vivo. [Mac]

UltraEdit

Otro editor que ha existido durante años. Este es un editor de texto inmensamente sólido y potente, capaz de abrir archivos limitados sólo por la cantidad de memoria del equipo. La lista de características es demasiado grande para enumerarla, con una enorme lista de funciones de manipulación de texto, soporte de proyectos, potente búsqueda y reemplazo, edición hexadecimal, listas de funciones, una enorme lista de idiomas soportados (600+), ftp para archivos remotos, telnet, ssh, comparación de archivos, macros personalizables, soporte de herramientas y compilador y mucho, mucho más. UltraEdit. [Linux, Mac, Windows]

Sublime Text

Un editor relativamente nuevo, Sublime Text es un nuevo enfoque en editores de código. "Abrir Cualquier Cosa" busca a través de nombres de archivo y el contenido de los archivos, con notable eficiencia. Controles de selección increíblemente poderosos permiten editar texto en múltiples ubicaciones a la vez y el "Minimapa" le da una vista de vuelo de pájaro del archivo abierto para que pueda encontrar su posición fácilmente. Activamente desarrollado, se agregan nuevas características contínuamente y la comunidad alrededor del editor se está expandiendo rápidamente. Macros, auto-completado, snippets (fragmentos de código), herramientas de compilación, la lista de funciones crece y crece. Soporta Linux y Mac a partir de la versión 2. [Linux, Mac, Windows]

Vim

Si tiene que preguntar, probablemente no es para usted. Vim [Linux, Mac, Windows]

Extensiones de Google Chrome

Herramientas de Desarrollador

No es realmente una extensión para Chrome, sino una herramienta ya incorporada (comparte mucho con el Inspector Web de Safari, ambos derivados de WebKit). Este conjunto de herramientas incluye un inspector del DOM, un depurador básico de JavaScript, herramientas de generación de perfiles, un inspector de carga de red y líneas de tiempo, inspectores de recursos de página y más. Herramientas de Desarrollador.

code cola

Un panel emergente con herramientas de edición de CSS para examinar y modificar los estilos de una página determinada. code cola.

Firebug Lite para Google Chrome

Realmente no necesita instalar una extensión para utilizar Firebug Lite con Chrome, aunque la extensión es agradable porque permite la aplicación con un sólo clic del script de Firebug Lite a cualquier página con la que esté trabajando. No tiene el conjunto completo de funcionalidades de Firebug, pero está cercano.

HTML5 Outliner

El HTML5 Outliner añade una ventana emergente con un esquema HTML5 generado de la jerarquía de encabezados de la página actual. Ayuda a comprobar la organización de sus páginas con el nuevo algoritmo de encabezados de HTML5.

Pendule

Bonito conjunto de herramientas para mostrar e interactuar con datos sobre la página web actual. Muy similar a la Web Developer Toolbar para Firefox original. Pendule.

Web Developer para Chrome

Chris Pederick, el desarrollador original de la Web Developer Toolbar original para Firefox ha portado la mayoría de la misma a Chrome. Aquí la tiene. Web Developer.

Extensiones de Firefox

FireFTP

FireFTP es un cliente FTP multiplataforma, seguro y gratis para Mozilla Firefox que proporciona acceso fácil e intuitivo a servidores FTP.

Firebug

Firebug se integra con Firefox para poner una gran cantidad de herramientas de desarrollo al alcance de su mano mientras navega. Puede editar, depurar y monitorear CSS, HTML y JavaScript en vivo en cualquier página web.

Firequery

Firequery es una colección de mejoras de Firebug para jQuery integrado en Firebug.

Firecookie

Firecookie agrega visualización, edición y eliminación de cookies a Firebug.

CSS Usage

CSS Usage es una extensión para Firebug que le permite escanear varias páginas de su sitio para ver que reglas CSS están siendo utilizadas realmente en su sitio.

Greasemonkey

Le permite personalizar la forma en que una página web se muestra con pequeños trozos de JavaScript. GreaseMonkey.

Web Developer Toolbar

Agrega un menú y una barra de herramientas con diversas herramientas de desarrollador web. Web Developer Toolbar.

JSView

Agrega un elemento en la barra de estado que muestra todos los archivos JavaScript y CSS externos cargados en una página determinada. Le permite hacer clic y ver los archivos y cosas como sus URLs. Es una gran manera de obtener las direcciones URL de los archivos y ponerlos en Charles para la depuración remota. JSView.

Live HTTP headers

Cuando se ejecuta, captura todo el tráfico HTTP desde el navegador, lo que le permite ver qué archivos están siendo solicitados, así como información sobre las solicitudes y respuestas del servidor. Live HTTP Headers.

Quick Locale Switcher

De gran ayuda cuando trabaja con internacionalización, Quick Locale Switcher le permite cambiar la configuración regional enviada junto con el encabezado HTTP user-agent del navegador, diciéndole a los servidores que muestren el contenido en otros idiomas.

Screengrab

Screengrab se coloca en la barra de estado de Firefox, permitiéndole capturar, copiar o guardar capturas de pantalla de todo, desde una selección en una página web hasta toda la página, incluso las partes que se muestran "debajo del doblez."

Total Validator

Permite pasar su página con un sólo clic a través de un validador de etiquetado. ¡No hay mejor manera de comprobar rápidamente si faltan o no coinciden las etiquetas! También está disponible como una aplicación independiente. Total Validator.

Extensiones de Opera

Dragonfly es la herramienta para desarrolladores de Opera similar a Firebug.

Plugins de IE

CompanionJS, DebugBar, IE8 Dev tools.

Charles Proxy

Charles observa todas las solicitudes y puede darle un montón de información sobre ellas. También es supremamente útil Map Local, que le permite utilizar un archivo local en lugar de una dirección URL determinada (bueno para reemplazar un js minimizado por uno completo).

Fiddler

Citando al sitio original: "Fiddler es un Proxy de Depuración Web que registra todo el tráfico HTTP(S) entre su computador e Internet. Fiddler le permite inspeccionar el tráfico, establecer puntos de marca y "jugar" con los datos entrantes o salientes. Fiddler incluye un potente subsistema de secuencias de comandos basado en eventos y puede ampliarse utilizando cualquiera lenguaje .NET".

Speedlimit

Speedlimit es un panel de preferencias de Leopard (funciona en Snow Leopard) para limitar su ancho de banda de red a unas cuantas velocidades diferentes: 768k DSL, Edge, 3G y acceso discado. Es bueno para probar las más bajas velocidades que desea soportar o cuando quiere saber cómo funcionará su aplicación en velocidades reales.
